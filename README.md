### ``Directory 구조``

#### client
```

client
├── dist
├── src
│   ├── index.html
│   ├── main.js
│   ├── assets
│   ├── component // 페이지에 해당하는 컴포넌트들
│   │   ├── chat
│   │   ├── detail
│   │   ├── ...
│   │   └── shared // 공용으로 쓰이는 네비게이션바, 버튼 등의 컴포넌트
│   ├── core // 다른 컴포넌트들의 기반이 되는 컴포넌트
│   │   └── Component.js
│   ├── lib // 해시 라우터
│   │   └── router.js
│   ├── scss
│   ├── store // 전역 상태객체
│   │   └── index.js 
│   └── utils.js
└── webpack.config.js

```
